# LAKI Principles

These principles guide public LAKI documentation, examples, review workflows, and future data-quality work.

## Lao-Native First

LAKI starts from Lao language needs, Lao usage, Lao context, and Lao readers. English documentation may help collaboration, but the project goal is stronger Lao-native language infrastructure.

## Human Review Before Acceptance

LAKI is human-led. AI tools may assist with drafting, testing, and analysis, but reviewed outputs are not accepted only because they were generated by AI. Final acceptance requires human review, source checking, and validation.

## Source Authority Matters

Data quality depends on where information comes from. Official, institutional, academic, published, community, AI-generated, and unknown sources should not be treated as equal.

## Raw Source Preservation

When source material is safe and allowed to be used, review notes should preserve enough provenance to understand what was checked. Public records should avoid exposing restricted, private, or sensitive raw material.

## No Blind AI-Generated Data Acceptance

AI-generated candidate data is never automatically trusted. It may be useful as a draft, test case, or candidate, but it must pass review before acceptance.

## Small Reviewed Packs Before Large Datasets

LAKI prefers small, carefully reviewed packs before large unverified datasets. Quality, traceability, and correction history matter more than size.

## No Private or Sensitive Data in Public Repo

This public repository must not contain private code, raw private data, review CSVs, finance parser rows, internal datasets, API keys, credentials, personal data, government-sensitive materials, or anything from a `data/` folder.

## Clear Uncertainty

When something is not verified, it should be marked clearly. Uncertainty is safer than unsupported confidence.

## Practical Local Usage Matters

LAKI should support useful Lao-language work: OCR, search, parsing, education, research, local applications, public-interest documentation, and better evaluation of AI outputs.
