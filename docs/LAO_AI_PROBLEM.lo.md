# Lao AI Quality Problem
ບັນຫາຄຸນນະພາບ AI ພາສາລາວ

ເອກະສານນີ້ອະທິບາຍບັນຫາທີ່ພົບໄດ້ໃນ AI, OCR, parser, ແລະຂໍ້ຄວາມພາສາລາວທີ່ຖືກສ້າງຂຶ້ນ. ນີ້ບໍ່ແມ່ນການໂຈມຕີບຸກຄົນ, ອົງການ, ຜະລິດຕະພັນ, ຫຼືໂຄງການໃດໆ.

## Common Issues
ບັນຫາທີ່ພົບເລື້ອຍ

### ໂຄງສ້າງທີ່ໄດ້ອິດທິພົນຈາກພາສາອື່ນ

ຂໍ້ຄວາມພາສາລາວບາງຄັ້ງມີລໍາດັບຄໍາ ຫຼືວິທີຂຽນທີ່ບໍ່ຄ່ອຍເປັນທໍາມະຊາດສໍາລັບຄົນລາວ.

### ພາສາລາວຄ້າຍການແປ

AI ອາດສ້າງພາສາລາວທີ່ເໝືອນແປຕົງຈາກພາສາອັງກິດ ຫຼືພາສາອື່ນ. ອ່ານເຂົ້າໃຈໄດ້, ແຕ່ອາດແຂງ ຫຼືບໍ່ຄືຄົນລາວຂຽນ.

### ບໍລິບົດທ້ອງຖິ່ນອ່ອນ

AI ອາດບໍ່ເຂົ້າໃຈຊື່ທ້ອງຖິ່ນ, ຄໍາທາງລັດຖະການ, ບໍລິບົດການສຶກສາ, ວັດທະນະທໍາ, ການໃຊ້ຄໍາໃນຕະຫຼາດ, ຫຼືຄໍາເວົ້າປະຈໍາວັນ.

### ຄໍາສະກົດ ແລະຄໍາໃຊ້ບໍ່ສອດຄ່ອງ

ຂໍ້ມູນພາສາລາວອາດມີຮູບສະກົດ ແລະ spacing ຫຼາຍແບບ. AI ອາດປົນຮູບຄໍາໂດຍບໍ່ອະທິບາຍວ່າແບບໃດເໝາະ, ແບບໃດເກົ່າ, ຫຼືແບບໃດຜິດ.

### ບໍ່ມີແຫຼ່ງອ້າງອີງຊັດເຈນ

ຂໍ້ມູນທ້ອງຖິ່ນທີ່ AI ສ້າງມັກບໍ່ມີແຫຼ່ງອ້າງອີງ. ສໍາລັບວຽກພາສາລາວ, ຂໍ້ຄວາມທີ່ຂຽນດີຢ່າງດຽວບໍ່ພໍ. ແຫຼ່ງອ້າງອີງ ແລະ review status ສໍາຄັນ.

### ຂໍ້ມູນທ້ອງຖິ່ນທີ່ AI ສ້າງຂຶ້ນເອງ

AI ອາດສ້າງຊື່ອົງການ, ລາຍລະອຽດສະຖານທີ່, ສະຖິຕິ, ຫຼືຂໍ້ອ້າງດ້ານນະໂຍບາຍທີ່ບໍ່ມີຈິງ. ຄວາມຜິດແບບນີ້ອາດດູໝັ້ນໃຈ ແລະຂຽນສວຍ.

### OCR ຜິດ

OCR ພາສາລາວອາດສັບສົນຕົວອັກສອນ, ວັນນະຍຸດ, ສະຫຼະ, spacing, line break, ແລະ mixed-script text. OCR output ຄວນຖືເປັນ candidate text ຈົນກວ່າຈະກວດແລ້ວ.

### model ຕອບໝັ້ນໃຈເກີນໄປ

AI ມັກຕອບຢ່າງໝັ້ນໃຈ ແມ່ນແຕ່ເມື່ອ input ບໍ່ຄົບ, source ອ່ອນ, ຫຼືຂໍ້ຄວາມພາສາລາວຍັງບໍ່ໄດ້ກວດໂດຍຜູ້ຮູ້ພາສາ.

## LAKI Response
ວິທີຕອບຂອງ LAKI

LAKI ເນັ້ນ source authority, human review, uncertainty ທີ່ຊັດເຈນ, ຕົວຢ່າງນ້ອຍທີ່ກວດແລ້ວ, OCR evaluation, parser validation, ແລະ methodology ທີ່ປອດໄພສໍາລັບ public repo.
