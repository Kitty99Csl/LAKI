# LAKI ແມ່ນຫຍັງ?

LAKI ຫຍໍ້ມາຈາກ Lao AI Knowledge Infrastructure.

LAKI ແມ່ນໂຄງສ້າງພື້ນຖານ, ບໍ່ແມ່ນ chatbot ຢ່າງດຽວ. ມັນເປັນພື້ນທີ່ສາທາລະນະສໍາລັບ documentation ແລະ methodology ເພື່ອປັບປຸງພື້ນຖານທີ່ AI ພາສາລາວຕ້ອງອາໄສ: ຄຸນນະພາບຂໍ້ມູນ, ການກວດແຫຼ່ງອ້າງອີງ, ການກວດໂດຍມະນຸດ, ການປະເມີນ OCR, parser logic, benchmark, ແລະ workflow ຄວາມຮູ້ທີ່ປອດໄພ.

## ໂຄງສ້າງພື້ນຖານ, ບໍ່ແມ່ນພຽງຜົນລັບ

ໂຄງການ AI ຫຼາຍອັນຖືກມອງຜ່ານຄໍາຕອບສຸດທ້າຍທີ່ model ໃຫ້. LAKI ເນັ້ນຂັ້ນຕອນກ່ອນນັ້ນ:

- ຂໍ້ມູນມາຈາກແຫຼ່ງໃດ?
- ພາສາລາວເປັນທໍາມະຊາດ ແລະເໝາະກັບບໍລິບົດລາວບໍ?
- ຂໍ້ຄວາມມາຈາກ OCR, parser, ມະນຸດ, ຫຼື AI?
- review status ແມ່ນຫຍັງ?
- ສາມາດຕິດຕາມ, ແກ້ໄຂ, ຫຼືປະຕິເສດໄດ້ບໍ?
- ປອດໄພທີ່ຈະເຜີຍແຜ່ບໍ?

## LAKI ສາມາດຊ່ວຍຫຍັງໄດ້

- AI tools ທີ່ຕ້ອງການຂໍ້ມູນພາສາລາວທີ່ດີຂຶ້ນ
- language models ທີ່ຕ້ອງການ evaluation examples ພາສາລາວທີ່ປອດໄພ
- OCR systems ທີ່ຕ້ອງການ error analysis ສໍາລັບອັກສອນລາວ
- parsers ທີ່ຕ້ອງການ input/output examples ທີ່ກວດແລ້ວ
- research datasets ທີ່ຕ້ອງການ provenance ແລະ status fields
- local applications ທີ່ຕ້ອງການຄວາມຮູ້ພາສາລາວທີ່ອີງແຫຼ່ງອ້າງອີງ
- benchmarks ທີ່ແຍກຄວາມລື່ນໄຫຼຂອງພາສາອອກຈາກຂໍ້ມູນທີ່ຢືນຢັນໄດ້

## ເປັນຫຍັງສໍາຄັນ

AI ພາສາລາວທີ່ດີບໍ່ໄດ້ມີແຕ່ການສ້າງຂໍ້ຄວາມ. ມັນຕ້ອງມີການກວດແຫຼ່ງອ້າງອີງ, review status, ຄວາມບໍ່ແນ່ນອນ, ຄໍາສະກົດ, ການໃຊ້ຄໍາ, OCR errors, ແລະບໍລິບົດທ້ອງຖິ່ນ.

LAKI ມອງຂໍ້ມູນພາສາລາວເປັນສິ່ງທີ່ຕ້ອງການການກວດ ແລະການດູແລ, ບໍ່ແມ່ນພຽງ output ຊົ່ວຄາວຈາກ model.
